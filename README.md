# HR Analytics Dashboard

# Overview
This project presents an interactive HR analytics dashboard built using Power BI to analyze employee demographics, attrition trends,
hiring patterns, and workforce performance.

# Tools Used
- Power BI


# Dashboard Features
- Employee demographics analysis
- Attrition and retention insights
- Department-wise employee distribution
- Hiring trends and workforce overview
- Interactive KPI visualizations

# Key Insights
- Identified departments with higher attrition rates
- Analyzed workforce distribution across age groups and job roles
- Provided insights to support HR decision-making
